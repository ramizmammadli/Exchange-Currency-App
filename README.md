# Exchange Currency App.

### System Features.

System includes 3 features:

- "Show Currencies" -> By clicking this button, the system takes real-time TL (Turkish lira) equivalents of currencies via "doviz.com". By this way, User can see current rates without using any websites or exchange office!
- "Exchange" -> This button generates the feature of exchanging the currencies. By choosing two values from combo boxes, User can easily convert currencies into each other.
- "Close" -> To conclude, By clicking this button, User closes the application.
